## Что бы я взял первым (моё мнение)

В порядке value/effort:
1. **Auto-save в localStorage** — если браузер крэшнется во время рисования, сейчас потеряется всё.
2. **Hotkeys на инструменты** — банально, но круто экономит время.
3. **Brush size > 1** — без этого расфокусированно красить большие зоны мучительно.
4. **Bucket fill** — для быстрой заливки больших однородных областей.
5. **Map metadata + Recent files** — даёт workspace-feel вместо одного безымянного файла.
6. **Resize map** — сейчас если выбрал не тот размер при создании, переделывать вручную муторно.
